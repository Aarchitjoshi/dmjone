Logic Gates and its verification in Proteus

1. Introduction to Logic Gates

In this experiment, we will verify the truth table for each basic logic gate using the Proteus software. We will use the following integrated circuits (ICs) for each logic gate:

  • AND gate: 7408
  • OR gate: 7432
  • NOT gate: 7404
  • NAND gate: 7400
  • NOR gate: 7402
  • XOR gate: 7486
  • XNOR gate: 74266

2. Proteus Setup

To verify the truth table for each gate in Proteus, follow these steps:

  1. Open Proteus software.
  2. Create a new project.
  3. Add the appropriate IC for each logic gate to the design.
  4. Connect input and output pins to switches and LED indicators, respectively.
  5. Power the circuit and test the truth table by toggling the switches.

Procedures of Doing the Experiment

AND Gate

Title: Verification of Truth Table of an AND Logic Gate in Proteus Software

Aim: To verify the truth table of an AND logic gate in Proteus software.

Requirements: Proteus software, 7408 AND gate IC, Logic State and Logic Probe tools.

Procedure

  1. Open Proteus software and create a new schematic.
  2. Add 7408 AND gate IC to the schematic.
  3. Place the Logic State tool to provide inputs to the AND gate.
  4. Place the Logic Probe tool to observe the output of the AND gate.
  5. Start the simulation and apply the input combinations "00", "01", "10", and "11".
  6. Observe the output values and compare them to the expected truth table for an AND gate.

Conclusion: The observed results match the expected truth table, validating the functionality of an AND gate in Proteus software.
