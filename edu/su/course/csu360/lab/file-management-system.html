File Management System

1. Overview of File Management System

File Management System (FMS) is an essential component of an operating system that organizes, stores, retrieves, and manages data files on a computer. This practical involves using command-line tools in Kali Linux to perform basic file operations.

1.1 Creating, Listing, and Removing a File

Create a file, list it, and then remove it to demonstrate basic file operations:

# Create a new file
touch example.txt
# List the file
ls -l
# Remove the file
rm example.txt

1.2 Working with Directories

Create a directory, list it, and then remove it:

# Create a new directory
mkdir dir
# List directories
ls -l
# Remove the directory
rmdir dir
# List directories to check if directory removed
ls -l

1.3 Copying and Moving Files

Create a file, copy it to a new file, move the new file to a new name, and then clean up:

# Create a new file
touch original.txt
# Copy the file to a new file
cp original.txt copy.txt
# Move the copied file to a new name
mv copy.txt moved.txt
# Remove both files
rm original.txt moved.txt

1.4 Directory Operations with Content

Create a directory with a file in it, copy the directory, move the copied directory, and then remove everything:

# Create a directory and a file within it
mkdir testDir
touch testDir/file.txt
# Copy the directory recursively
cp -r testDir copiedDir
# Move the copied directory
mv copiedDir movedDir
# Remove directories and their contents
rm -r testDir movedDir

1.5 Viewing File Content

Create a file, add text, view the content, and then remove the file:

# Create a new file
echo "Hello World!" > viewfile.txt
# Display the content of the file
cat viewfile.txt
# Remove the file
rm viewfile.txt
